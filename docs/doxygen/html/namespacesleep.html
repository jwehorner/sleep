const void sleep::sleep_ms_corrected ( const uint32_t  ms,
const int64_t  error_us 
)

Function sleep_ms_corrected sleeps for the specified number of milliseconds minus the provided schedule slip, so that the mean sleep time converges onto the desired period.

Parameters
msuint32_t number of milliseconds to sleep for.
error_usint64_t number of microseconds of error accumulated by sleeping.

This function is intended to be used in the included kind of error tracking pattern, where the schedule slip is tracked with each iteration, so when the slip gets too large a sleep is skipped and the schedule catches back up.

int64_t start_time_us, error_us = 0;
uint32_t duration_ms = 1;
while(CONDITION) {
start_time_us = sleep::now_us();
sleep::sleep_ms_corrected(duration_ms, error_us);
error_us += ((int64_t)sleep::now_us() - start_time_us) - (duration_ms * 1'000);
}
